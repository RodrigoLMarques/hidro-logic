## Sobre o Projeto

HidroLogic é um sistema digital de monitoramento do consumo de água residencial, simulado no WiredPanda. Relaciona-se ao ODS 6 (Água Potável e Saneamento) como foco principal, e aos ODS 11 e 12.

## Como Rodar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/RodrigoLMarques/hidro-logic.git
```

2. Para rodar o projeto é preciso utilizar o programa wiRedPanda

3. O arquivo principal do projeto é ```main.panda```
